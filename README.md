# Bookshelf

A book discovery website built with HTML, CSS and JavaScript.
Search for books, save them to your reading list, and explore by subject.

Book data comes from the [Open Library API](https://openlibrary.org) — free, no API key needed.

## Live Site

https://badal-khunt.github.io/bookshelf-app

## Pages

- **Home** — landing page with featured books
- **Search** — search and filter books by subject and sort order
- **Reading List** — your saved books
- **Contact** — contact form with validation
- **Privacy** — GDPR privacy policy

## How to Run

Open in VS Code and use Live Server — right-click `index.html` → Open with Live Server.

## Built With

- HTML, CSS, JavaScript (no frameworks)
- Open Library API
